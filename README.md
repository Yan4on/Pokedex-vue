# Pokedex-vue
### Первый проект для самостоятельного изучения фреймворка Vue.
- Все данные должны браться с https://pokeapi.co/

### Обзор

 **Описание проекта**: 
 - Отображение случайной карточки
 - Получение информации о карточке
 - Поиск по имени карточки
 - Адаптивность
 

**Техноогии проекта**: 

- Vue
- JavaScript
